This repo contains the MC68040 local bus card project for the [AmigaPCI](https://github.com/AmigaPCI). It supports up to 128MB of fast ram.  

The documentation at the [AmigaPCI](https://github.com/AmigaPCI) project should be reviewed before building this project.  Technical information on the AmigaPCI Local Bus Port can be found in the [AmigaPCI Hardware Reference Guide](https://github.com/AmigaPCI/AmigaPCI-MainBoard/blob/main/AmigaPCI_Hardware_Reference_Guide.md)
